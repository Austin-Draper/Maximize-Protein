# Maximize-Protein
THIS PROGRAM WAS UPLOADED TO GITHUB USING VISUAL STUDIO 2015 COMMUNITY EDITION

Compute the set of foods that maximizes protein, within a calorie budget, with the exhaustive search method or dynamic programming.

To run this program, there are two options.
1. Open the .sln in the same version of visual studio and run the program from there.
2. Navigate to the cs 335 project 3 folder and run the executable.
